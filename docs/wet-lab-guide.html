TurboCell Atlas

What to put in, what comes back, and how to read it.

This page is the easy entry point for wet researchers. It uses executed analyses on the SCimilarity tutorial dataset and explains the outputs in plain language before asking you to care about the algorithm.

In one sentence

If I describe one cell state that I care about, what similar cells can I find in a large atlas?

What you put in

A reference atlas, a query state such as a centroid of annotated cells, and optional filters such as disease or tissue.

What you get back

A ranked list of similar cells, summary tables for interpretation, and figures that compare methods or search settings.

The easiest starting files are configs/wetlab_metadata_template.csv and notebooks/wet_lab_walkthrough.ipynb.
