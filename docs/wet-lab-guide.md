# Wet Lab Guide

This is the easiest reading path for wet-lab collaborators.

## What you put in

- a reference atlas represented as embeddings plus metadata
- one query state, often a centroid or one interesting cell
- optional filters such as disease or tissue

## What you get back

- a ranked table of similar cells
- summary tables for biological interpretation
- figures showing what changed across methods or settings

## Start with these files

- `configs/wetlab_metadata_template.csv`
- `notebooks/wet_lab_walkthrough.ipynb`
- `artifacts/wetlab_examples/`
