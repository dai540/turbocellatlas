Get Started

Run your first TurboCell Atlas search.

This page gets the package running from local embedding files with the smallest practical setup.

Install

pip install -e .

pip install -e .[io,bench,dev]

io enables .h5ad loading, bench enables HNSW and FAISS-PQ, and dev enables tests.

Prepare input files

Use a reference embedding bank in .npy or .npz, a metadata table in .jsonl, .json, or .csv, and a query vector in .npy.

{"cell_id": "cell-0001", "study": "ILD-demo", "disease": "fibrosis", "tissue": "lung"}

The easiest starting files are configs/wetlab_metadata_template.csv and notebooks/wet_lab_walkthrough.ipynb.

CLI example

tca search ^
  --embeddings data/reference.npy ^
  --metadata data/reference.jsonl ^
  --query data/query.npy ^
  --bit-width 3 ^
  --candidate-k 128 ^
  --rerank-k 20 ^
  --output artifacts/results.json

You can add metadata filters such as --filter disease=fibrosis and --filter tissue=lung.

Python API example

import numpy as np
from tca import SearchIndex, TurboQuantConfig

embeddings = np.load("data/reference.npy").astype(np.float32)
query = np.load("data/query.npy").astype(np.float32)[0]
metadata = [{"cell_id": "cell-0", "disease": "fibrosis"}]

index = SearchIndex.from_embeddings(
    embeddings=embeddings,
    metadata=metadata,
    config=TurboQuantConfig(bit_width=3, candidate_k=128, rerank_k=20, quantizer_kind="prod", seed=7),
)

results = index.search(query, filters={"disease": "fibrosis"})

Wet-lab shortcut

If you are not primarily a computational user, the shortest path is: read Wet Lab Guide, copy the metadata shape from configs/wetlab_metadata_template.csv, open notebooks/wet_lab_walkthrough.ipynb, and compare your first output with the example files in artifacts/wetlab_examples/.
