Main modules

tca.config holds TurboQuantConfig.

tca.quantization holds TurboQuantMSE, TurboQuantProd, and exact_topk.

tca.pipeline holds SearchIndex.

tca.io handles local embedding and metadata loading.

tca.benchmarks provides exact, HNSW, and FAISS-PQ comparison backends.

CLI

tca search --embeddings REF.npy --metadata META.jsonl --query QUERY.npy --output results.json

Important options: --bit-width, --candidate-k, --rerank-k, --oversample, --quantizer-kind, and repeated --filter key=value.

Current limits

The package currently assumes dense in-memory embeddings and exact reranking from the original bank. Direct SCimilarity model execution and Census ingestion are planned next layers rather than finished parts of the current codebase.
