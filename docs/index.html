Atlas retrieval docs built around workflows, evidence, and reviewability.

TurboCell Atlas compresses candidate generation with TurboQuant and restores final ranking with exact reranking in the original embedding space. The docs now reflect how people actually use the system: getting started, following scenario tutorials, reading design rationale, and inspecting executed benchmark artifacts.
