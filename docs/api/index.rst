API reference
=============

The intended public package surface is deliberately small. Most users should start from the high-level search index and configuration types rather than low-level internals.

Public entry points
-------------------

* ``tca.pipeline.SearchIndex``
* ``tca.config.TurboQuantConfig``
* ``tca.quantization.TurboQuantMSE``
* ``tca.quantization.TurboQuantProd``
* ``tca.cli``
