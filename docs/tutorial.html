Tutorials

Tutorials should explain not only what to run, but also what the result means.

Use this section for first runs, wet-lab-friendly walkthroughs, and scenario-specific executed examples. Each page is meant to connect a biological question, a query definition, and a concrete output table or figure.

How to use this section

A good TurboCell Atlas tutorial is more than a command log. It should help you understand why a query was chosen, what data are being searched, what tables and figures were generated, and how to interpret the result without over-claiming. If you are new to the package, read the two simple articles below before choosing a scenario tutorial.

How to read a TurboCell Atlas tutorial

Start from the background and objective, then inspect the input description, then look at the output tables and figures, and only after that read the interpretation. The package is most useful when the retrieval result can be traced back to a clear biological question and a clearly described input dataset.
