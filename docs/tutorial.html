Why this structure changed

One tutorial was hiding the actual ways people use the system.

TurboCell Atlas is not only a package API. It is a retrieval workflow. Some readers need to understand why rare-state retrieval works, some need to apply metadata filters, and some need to judge whether the benchmark artifacts are convincing enough to trust.
