# API

TurboCell Atlas exposes a small core API.

## Main classes

- `tca.pipeline.SearchIndex`
- `tca.config.TurboQuantConfig`
- `tca.quantization.TurboQuantMSE`
- `tca.quantization.TurboQuantProd`

## Main workflows

- build an index from embeddings and metadata
- search with optional metadata filters
- compare exact search with benchmark backends
- run the CLI for file-based retrieval
