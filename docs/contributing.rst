Contributing
============

TurboCell Atlas is still an actively shaped research codebase. Contributions are welcome, but the most helpful contributions are the ones that preserve traceability.

Good contributions usually do one or more of the following:

* improve retrieval behavior or benchmark traceability
* clarify documentation for wet-lab or collaborative users
* add tests for new query or filtering behaviors
* improve reproducibility of the public demo asset workflow

Before opening a change, please read the repository-level contribution guide in ``CONTRIBUTING.md`` and make sure that benchmark-facing changes keep the artifact story readable from input to output.
